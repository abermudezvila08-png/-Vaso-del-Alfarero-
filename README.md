# Vaso del Alfarero

## Visión del Proyecto
Vaso del Alfarero es una plataforma diseñada para fomentar la comunidad y el intercambio de conocimientos a través de espacios de colaboración. Nuestro objetivo es proporcionar un entorno donde los usuarios puedan interactuar, aprender y compartir experiencias.

## Características
- **Autenticación de Usuarios**: Un sistema robusto para la gestión de usuarios, con autenticación segura.
- **Interacción Comunitaria**: Espacios para que los usuarios se conecten y compartan información valiosa.
- **Gestión de Quizzes**: Funcionalidad para la creación y administración de quizzes interactivos.
- **Mensajería Interna**: Sistema de mensajería para facilitar la comunicación entre usuarios.
- **Tienda**: Una sección dedicada a la compra y venta de productos relacionados con la cultura y el arte.

## Stack Técnico
- **Backend**: Node.js con Express para la creación de APIs.
- **Base de Datos**: MongoDB para el almacenamiento y gestión de datos.
- **Frontend**: Flutter para la aplicación móvil, brindando una experiencia nativa en múltiples plataformas.
